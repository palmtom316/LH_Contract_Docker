# 合同查询机器人功能说明

## 功能概述

新增的**合同查询机器人**功能允许用户通过输入合同序号或合同名称进行模糊查询，快速获取上游合同及其关联的下游合同、管理合同和无合同费用的完整财务信息。

## 使用方法

1. 登录系统后，在页面右下角可以看到一个蓝色的**搜索图标按钮**
2. 点击按钮，弹出"合同查询助手"对话框
3. 在搜索框中输入：
   - 合同序号（如：1、2、10）
   - 合同名称的任意部分（如：装修、建设）
   - 合同编号的任意部分
4. 点击"查询"按钮或按回车键
5. 系统返回匹配的上游合同列表，点击展开可查看详细信息

## 查询结果内容

### 上游合同信息
- 合同序号
- 合同编号
- 合同名称
- 甲方/乙方
- **签约金额**
- **应收款**（应付款方的应付）
- **挂账金额**
- **已收款**

### 关联下游合同
列表显示所有关联的下游合同，包含：
- 合同序号
- 合同名称
- **签约金额**
- **应付款**
- **挂账金额**
- **已付款**
- 合计汇总

### 关联管理合同
列表显示所有关联的管理合同，包含：
- 合同序号
- 合同名称
- **签约金额**
- **应付款**
- **挂账金额**
- **已付款**
- 合计汇总

### 无合同费用
按费用分类显示关联的无合同费用（包括零星用工），显示：
- 管理费
- 培训费
- 餐饮费
- 交通费
- 咨询费
- 业务费
- 租赁费
- 资质费
- 车辆使用费
- 零星用工
- 其他费用

## 技术实现

### 后端
- **API 端点**: `GET /api/v1/contracts/search`
- **文件**: `backend/app/routers/contract_search.py`
- **参数**:
  - `query` (必填): 搜索关键词
  - `limit` (可选): 返回结果数量限制，默认10，最大50

### 前端
- **组件**: `frontend/src/components/ContractQueryBot.vue`
- **API模块**: `frontend/src/api/contractSearch.js`
- **集成**: 在 `Layout.vue` 中引入，全局可用

## 部署说明

代码已添加到现有项目中，部署时只需重新构建即可：

```bash
# 重新构建后端和前端
docker-compose -f docker-compose.prod.yml build

# 重启服务
docker-compose -f docker-compose.prod.yml up -d
```

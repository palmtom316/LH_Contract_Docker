## 五、注意事项

### 5.1 数据安全

- ⚠️ 请妥善保管您的账号密码
- ⚠️ 定期修改密码，建议每3个月更换一次
- ⚠️ 离开工位时请退出系统或锁定电脑
- ⚠️ 删除操作不可恢复，请谨慎操作

### 5.2 文件上传

- 📎 支持格式: PDF, Excel, Word, 图片
- 📎 单文件最大: 50MB
- 📎 建议合同文件使用 PDF 格式

### 5.3 报表缓存

- 📊 报表数据有5-10分钟缓存
- 📊 如需最新数据，请点击【刷新】按钮
- 📊 导出的Excel为实时数据
